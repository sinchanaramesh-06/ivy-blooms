Ivy Blooms is a mini project for a simple indoor plant shopping website.

## Technologies Used
- HTML, CSS, JavaScript
- MySQL (phpMyAdmin)
- PHP
- Node.js
- React

## Project Structure
- frontend-html: Static website pages
- node-backend: Node.js backend server
- react-frontend: React version of the landing page
- database: SQL file for database structure and data

## How to Run
- Frontend HTML: Open index.html or run via XAMPP
- Database: Import the SQL file using phpMyAdmin
- Node backend: Run `node server.js`
- React app: Run `npm start`

## Note
This repository contains source code only. Database and servers are run locally.
